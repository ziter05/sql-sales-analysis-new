# SQL Sales Analysis Project

## 📌 Opis
Prosty projekt SQL pokazujący analizę danych sprzedażowych.

## 🧱 Struktura
- schema.sql – struktura bazy
- insert_data.sql – dane testowe
- queries.sql – zapytania analityczne

## 🎯 Przykładowe analizy
- całkowita sprzedaż
- top klienci
- najpopularniejsze produkty

## 🛠 Technologie
- SQL (PostgreSQL / MySQL)

## 🚀 Jak uruchomić
1. Uruchom schema.sql
2. Dodaj dane z insert_data.sql
3. Odpytaj queries.sql
